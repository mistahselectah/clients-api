## Clients API

### Требования к системе
- Node.js v20.11.1
- docker-compose v2.28.1


### Установка и запуск
```
git clone https://github.com/mistahselectah/clients-api.git
cd clients-api
cp .env.sample .env
docker-compose up -d
```

Документация по адресу http://localhost:<API_PORT>

PgAdmin по адресу http://localhost:<PGADMIN_PORT>
